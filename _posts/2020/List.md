# List

리스트와 연결리스트는 다른 개념. 

List 구현방법에 따라 두 가지로 나눌 수 있다. 

- 순차 리스트 : 배열을 기반으로 구현된 리스트 
- 연결 리스트 : 메모리의 동적 할당을 기반으로 구현된 리스트 
  
<br/>
<br/>

> ### LIST의 기본적이고 중요한 특성 
리스트의 ADT를 정의할 때 고려애야할 요소들
 - 데이터를 나란히 저장한다. 
 - 데이터 중복 저장이 가능하다.
